# EAT: Fake News Detection with Emotion and Adversarial Training

## 📌 Introduction

This project implements a fake news detection model based on:

* Emotion features
* Adversarial training
* Dynamic fusion mechanism

---

## 🧠 Model Overview

The proposed EAT model improves robustness and generalization ability.
